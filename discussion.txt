Command-line Interface
- CLI
- allows us to interact with computer programs by typings
commands rather using graphical user interface (GUI)
 
CLI interface command
 
1. cd - Change Directory(move or change from the existing directory
or folder to the desired directory)
ex. cd <directory/folder name>
2. mkdir - Create Folder( create a new directory/folder)
ex. mkdir <directory/folder name>fin
3. ls - List (show/display all the files and folders within the given path)
ex. ls
4. touch - Touch(create a new fil)
ex. touch <filename.extension>
5. rm - Remove (remove a chosen file)
ex. rm <filename.extension>
6. cat - Display(display the content of a file)
ex. cat <filename.ext>
7. find - Find(search a file)
ex. file . -name "filename.txt"
